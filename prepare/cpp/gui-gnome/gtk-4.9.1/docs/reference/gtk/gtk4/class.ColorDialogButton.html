unstable since: 4.10

Description [src] 

final class Gtk.ColorDialogButton : Gtk.Widget {
  /* No available fields */
}

The GtkColorDialogButton is a wrapped around a GtkColorDialog and allows to open a color chooser dialog to change the color.

It is suitable widget for selecting a color in a preference dialog.

CSS nodes

colorbutton
╰── button.color
    ╰── [content]

GtkColorDialogButton has a single CSS node with name colorbutton which contains a button node. To differentiate it from a plain GtkButton, it gets the .color style class.
